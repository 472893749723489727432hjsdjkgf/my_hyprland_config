# Как скачать?

* обновить

 ```bash
sudo pacman -Suy
```

* поставить все нужное

```bash
sudo pacman -S hyprlauncher kitty hyprland fastfetch wofi waybar  mpvpaper polkit-kde-agent

```

* скопировать репозиторий

```bash
git clone https://github.com/472893749723489727432hjsdjkgf/my_hyprland_config && cd hyprland_conf
```

* запустить python скрипт

```bash
python install.py
```

* желательно перезагрузиться
